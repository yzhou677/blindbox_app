# Documentation Archive

This folder preserves historical Shelfy documents.

Archived documents explain what happened in the past: completed implementation
phases, audits, release-specific plans, investigations, and generated reports.
They are not the current source of truth for product semantics or implementation
contracts.

Use:

- [`../decisions/`](../decisions/) for durable ADRs and PDRs
- [`../README.md`](../README.md) for the current documentation index
- active architecture docs for how Shelfy works now
